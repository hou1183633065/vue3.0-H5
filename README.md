# vue3.0
###安装vue3.0
npm install -g @vue/cli
### OR
yarn global add @vue/cli
###查看版本
vue --version

-进入项目运行vue ui 打开可视化界面进行配置

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

### Run your end-to-end tests
```
yarn run test:e2e
```

### Run your unit tests
```
yarn run test:unit
```
